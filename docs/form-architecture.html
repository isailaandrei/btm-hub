6. Summary Table

ConcernWhereHow
Form definition src/lib/academy/forms/*.ts Declarative config objects composed from shared + program-specific steps
Registration registry.ts + index.ts Side-effect imports auto-register into in-memory map
Schema generation schema-builder.ts Dynamic Zod — per-step for client, full + visibility-aware for server
Conditional fields isFieldVisible() visibleWhen rules evaluated in both UI and schema builder
Client validation page.tsx buildStepSchema → safeParse on "Next" click
Server validation actions.ts buildFullSchema → safeParse on submit
Persistence localStorage Keyed by slug, versioned by field hash, cleared on success
Submission actions.ts → Supabase Parse FormData → validate → find/create contact → insert application
